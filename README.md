# 🐾 Sistema de Clínica Veterinária

Sistema web simples para gerenciamento de uma clínica veterinária.

 ## 🚀 Tecnologias utilizadas

 Frontend
- Angular
- Tailwind

Backend
- Java 21
- Spring Boot

 Banco de Dados
- PostgreSQL

---

 ## 📌 Funcionalidades

- Criar nova conta 
- Recuperar minha senha
- Cadastro de animais
- Cadastro de clientes 

---

## 🏗️ Arquitetura do Projeto

O projeto está dividido em:

/backend → API REST em Spring Boot 

/frontend → Aplicação Angular 

/database → PostgreSQL integrado

---

## 🗄️ Banco de Dados

O banco de dados utilizado é PostgreSQL.

O arquivo `bank.sql` presente no repositório contém o backup do banco com as tabelas do sistema.

Para recriar o banco, basta importar o arquivo no PostgreSQL.
